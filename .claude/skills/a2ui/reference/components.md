# A2UI Component Reference

This file has been moved to MCP (Model Context Protocol) for dynamic component documentation.

## Using Component Documentation

To get information about A2UI components, use the MCP tools available to the agent:

- **list_components()** - List all available components
- **get_component(name)** - Get detailed documentation for a specific component
- **search_components(keyword)** - Search for components by keyword

## Standard Catalog Components

The A2UI 0.8 Standard Catalog includes 16 components across three categories:

**Layout Components:** Row, Column, List, Card, Tabs, Modal, Divider
**Display Components:** Text, Image, Icon, Video, AudioPlayer
**Input Components:** Button, TextField, CheckBox, DateTimeInput, MultipleChoice, Slider

Use the MCP tools above to get detailed documentation for any component.
